リレーショナルデータベース(RDB) 複数の表の形式でデータを管理する RDBの基本構造 それぞれの表をテーブル テーブルは列と行で構成される データベース管理システム(DBMS) DBMSはコンピュータ内で常に稼働してSQLを待ち受けており、 届いたSQLの内容に従って、データベースの内容を検索したり、書き換えたり する処理を実行する DBMSのうち、複数の表の形式でデータを取り扱うものをRDBMSという。 代表的なRDBMSは、MySQL,PostgreSQL,OracleDBなどがある SQL基礎 検索 SELECT カラム FROM テーブル名; 全てを選択するには * 条件をつけてより細かく検索するにはWHERE WHERE 条件文 例.WHERE 出金額 > 1000 追加 INSERT INTO テーブル名(カラム1, カラム2…) VALUES (データ1, データ2…); 更新 UPDATE テーブル名 SET カラム名 = 設定する値(変更する値) WHERE 条件; 削除 DELETE FROM テーブル名 WHERE 条件;
